Programming Language 73 [PL73]
==============================

Usage
=====
Examples for source code writing

### var - Variable declaration (int, float, string, boolean)
  
<b> Example: </b> <br /> <br />
	&ensp;int     --> var x = 1; <br />
	&ensp;float   --> var x = 1.2; <br />
	&ensp;String  --> var x = "testo"; <br />
	&ensp;boolean --> var x = true || var y = false; 
   
### input - Input from standard input
  
<b> Example: </b> <br /> <br />
	&ensp;input x; 
  
### println - Output on standard output
  
<b> Example: </b> <br /> <br />
	&ensp;println x; 
  
### main - Main method
  
<b> Example: </b> <br /> <br />
	&ensp;main <br />
    	&ensp;&ensp;1. ... ; <br />
    	&ensp;&ensp;2. ... ; <br />
    	&ensp;&ensp;3. ... ;             
    	&ensp;&ensp;4. ... ; <br />
	&ensp;end; 
  
### if - If, elif - Else-if, else - Else
 
<b> Example 1: </b> <br /> <br />
	&ensp;if x > y <br />
    	&ensp;&ensp;1. ... ; <br />
       	&ensp;&ensp;2. ... ; <br />
       	&ensp;&ensp;3. ... ;            
       	&ensp;&ensp;4. ... ; <br />
   	&ensp;end; <br /> <br /> <br />
 
<b> Example 2: </b> <br /> <br />
   	&ensp;if x > y <br />
   		&ensp;&ensp;1. ... ; <br />
       	&ensp;&ensp;2. ... ; <br />
       	&ensp;&ensp;3. ... ;            
       	&ensp;&ensp;4. ... ; <br />
   	&ensp;elif <br />
       	&ensp;&ensp;5. ... ; <br />
       	&ensp;&ensp;6. ... ; <br />
       	&ensp;&ensp;7. ... ;            
       	&ensp;&ensp;8. ... ; <br />
   	&ensp;end;
 
### for - For cicle
  
<b> Example: </b> <br /> <br />
   	&ensp;for 9 <br />
       	&ensp;&ensp;1. ... ; <br />
       	&ensp;&ensp;2. ... ; <br />
       	&ensp;&ensp;3. ... ;           
       	&ensp;&ensp;4. ... ; <br />
   	&ensp;end; 
  
### while - While   
  
<b> Example: </b> <br /> <br />
	&ensp;while x > y <br />
    	&ensp;&ensp;1. ... ; <br />
       	&ensp;&ensp;2. ... ; <br />
       	&ensp;&ensp;3. ... ;            
       	&ensp;&ensp;4. ... ; <br />
   	&ensp;end; 
  
### do_while - Do-while
  
<b> Example: </b> <br /> <br />
		&ensp;&ensp;1. ... ; <br />
    	&ensp;&ensp;2. ... ; <br />
       	&ensp;&ensp;3. ... ;            
       	&ensp;&ensp;4. ... ; <br />
   	&ensp;do_while x > y;
